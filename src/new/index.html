  • About Me:
    • I Love free (as in freedom) open source software and community driven projects. Some of my favorite software is made by Suckless Software and the GNU collection. I value privacy over everything, I despise subscription services. The modern digital world is a mess, having to navigate mass surveillance, censorship and the ever-increasing overabundance of artificial intelligence. Put the power back into the hands of the people.

    I've been using technology for as long as I can remeber, along with this I've always been a fan of unix based operating systems like Linux and BSD. Over the last few months I've started daily-driving FreeBSD for development with a dualbooted ArchLinux machine for bsd-non-compatable software like steam. I am familiar with majority of linux distros and now most of the "popular" BSD varients. some of these include, Ubuntu, Linux Mint, Pop_OS, Manjaro, Arch Linux, Crux Linux, Parabola Linux, Slackware, SalixOS, OpenBSD, DragonFly BSD, NetBSD, FreeBSD, Debian and more.
